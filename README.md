GigFlow is a mini freelance marketplace where users can post jobs (gigs) and freelancers can bid on them. The platform supports secure authentication and a complete hiring workflow.

## Features
- User registration and login with JWT authentication
- Create and view job postings
- Freelancers can place bids on gigs
- Clients can hire one freelancer per gig
- Automatic bid status update (hired / rejected)

## Tech Stack
- Frontend: React (Vite), Axios
- Backend: Node.js, Express.js
- Database: MongoDB Atlas
- Authentication: JWT (HttpOnly cookies)
